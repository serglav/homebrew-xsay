# homebrew-xsay

Homebrew tap for [xsay](https://github.com/serglav/xsay) — project-aware macOS TTS CLI.

## Install

```bash
brew install serglav/xsay/xsay
```

Or add the tap first:

```bash
brew tap serglav/xsay
brew install xsay
```

## Update

```bash
brew upgrade xsay
```

## Uninstall

```bash
brew uninstall xsay
```
